このネットワークで起きていること

入力
$X_0 \in \mathbb{R}^{N\times \text{id}}$, ターゲット $T$ (one-hot, $N\times \text{od}$)
順伝播
$X_l = f_l(X_{l-1} W_l + b_l)$  ($l = 1, 2, \ldots, L$)
$L$ = 層数 (入力以外の Linear 層の総数),   $X_L$ = 出力,   $f_l$ = 活性化関数
活性化
$f_l = \sigma$ (sigmoid),   $\sigma'(z) = X(1-X)$
損失
$\displaystyle E = \tfrac{1}{2}\|X_L - T\|_2^2$   MSE
逆伝播
$\delta_L = (X_L - T) \odot X_L \odot (1-X_L)$
$\delta_{l-1} = (\delta_l W_l^\top) \odot f_{l-1}'(X_{l-1})$
勾配
$\nabla W_l = X_{l-1}^\top \delta_l$,   $\nabla b_l = \mathbf{1}^\top \delta_l$
更新
$W_l \leftarrow W_l - \eta\, \nabla W_l$   (SGD, η = lr)
